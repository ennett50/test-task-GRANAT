Тестовое задание Granat

Для работы выполнить:

npm install -g gulp
cd ___gulp
npm install
gulp

Для задания сделана удобная структура расположения рабочей дирректории (вся 'магия' происходит в папке __dev) Также используются livereload, autoprefixer, uglify и другое.

Автоматизация: 1. Для шаблонов - jade 2. Для стилей - препроцессров stylus 3. Для js - coffeescript
